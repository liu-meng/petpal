# PetPal TabBar 优化设计方案（V1）

## 一、现状分析

```json
"tabBar": {
  "color": "#8A816F",
  "selectedColor": "#E08A2E",
  "backgroundColor": "#FFF8EE",
  "borderStyle": "white",
  "list": [
    { "pagePath": "pages/tasks/tasks",   "text": "任务" },
    { "pagePath": "pages/index/index",    "text": "宠物" },
    { "pagePath": "pages/parent/parent",  "text": "家长" }
  ]
}
```

**当前问题：**
1. ❌ 纯文字 tab，**无图标**，儿童识别成本高
2. ❌ 文案「任务 / 家长」**功能感太强**，不童趣
3. ❌ 配色 `selectedColor: #E08A2E` 偏暗，**选中态不够亮眼**
4. ❌ `borderStyle: "white"` 是原生 tabBar 的**默认白边**，视觉割裂
5. ❌ 无微交互反馈，点击**无视觉惊喜**

---

## 二、设计目标

| 目标 | 说明 |
|------|------|
| 🎯 降低认知成本 | 图标 > 文字，儿童一眼识别 |
| 🌈 增强童趣感 | 用 emoji/插画图标，不用线性图标 |
| 💛 强化选中反馈 | 选中态要有「情绪变化」 |
| 🏠 文案儿童化 | 「任务」→「做什么」之类 |
| ✨ 微交互惊喜 | 点击时图标有轻微动画 |

---

## 三、图标方案（核心）

微信小程序 tabBar **不支持直接放图片**（需用 `iconPath` / `selectedIconPath` 本地图片文件），因此有两种实现路径：

### 方案 A：原生 tabBar + 本地图标文件（推荐 ✅）

在 `assets/tab-icons/` 下放置 **6 张 81×81px PNG 图标**：

```
assets/
  tab-icons/
    tasks.png        ← 未选中：任务
    tasks-active.png ← 选中：任务
    pet.png          ← 未选中：宠物
    pet-active.png   ← 选中：宠物
    parent.png       ← 未选中：家长
    parent-active.png← 选中：家长
```

**图标设计风格要求：**
- 圆角、软萌、扁平插画风
- 未选中：灰色描边 + 浅色填充
- 选中：暖橙色实心 + 白色高光

---

### 方案 B：自定义 tabBar 组件（更灵活，但工作量更大）

用 `custom-tab-bar` 组件完全自定义，可实现：
- 图标动画（bounce / scale）
- 小红点提示
- 更丰富的选中态

**V1 建议先走方案 A**，验证效果后再升级方案 B。

| Tab | 当前 | 优化方案 A（童趣） | 优化方案 B（更童趣） |

| 宠物页 | 宠物 | 🐕 宠物 | 🏠 回家了 |

**推荐方案 A**（适度童趣，不过度低龄化）

## 五、配色优化方案

### 当前配色

| 状态 | 颜色 | 问题 |
|------|------|------|
| 未选中 | `#8A816F` | 偏灰，不够活泼 |
| 选中 | `#E08A2E` | 偏暗橙，不够亮眼 |
| 背景 | `#FFF8EE` | 偏黄，与页面背景接近 |
| 边框 | `white` | 白边在白色背景上**有割裂感** |

---

### 优化后配色（推荐）

| 状态 | 颜色 | 说明 |
|------|------|------|
| 未选中 | `#B5A68C` | 暖灰，柔和但不暗淡 |
| 选中 | `#FF8C42` | **亮橙色**，活泼吸睛 |
| 背景 | `#FFFDF6` | 接近白色，但带暖感 |
| 上边框 | `transparent` 或 `#F0E6D2` | 用**暖色细线**代替白边 |

**选中态图标配色建议：**
- 图标主色：`#FF8C42`
- 图标高光：`#FFD699`
- 背景光晕：径向渐变 `rgba(255,140,66,0.12)`

## 七、微交互设计（方案 B 自定义 tabBar 时实现）

如果后续升级为自定义 tabBar，可以加入以下微交互：

### 点击反馈

| 交互 | 效果 |
|------|------|
| 点击 tab | 图标 **scale(0.85) → 回弹 scale(1.0)**，持续 200ms |
| 切换选中 | 新选中图标 **bounce 动画**（类似 iOS spring） |
| 宠物 tab 选中 | 小心心 ❤️ **上浮消失动画** |

---

### 状态提示（红点）

| 场景 | 提示 |
|------|------|
| 有待审核打卡 | 家长 tab **显示红点** `+pendingCount` |
| 宠物状态变差 | 宠物 tab **轻微抖动**（sick 状态时） |
| 有任务待做 | 任务 tab **金色边框闪烁** |

## 九、图标制作规范

### 尺寸要求
- **81 × 81 px** @2x（微信推荐尺寸）
- 实际图标内容占 **60 × 60 px** 区域（留边距）
- 格式：**PNG-24，透明背景**

### 设计风格
```
整体风格：圆润、软萌、扁平插画
线条：圆角，无尖角
配色：选中态用暖色系（橙/黄）
阴影：可选，轻微投影增加层次
```

### 制作工具推荐
- **Figma**（推荐，有团队协作）
- **Sketch**
- **Canva**（快速出图）
- 或直接用 **emoji + 圆角背景** 快速出 MVP 版本
